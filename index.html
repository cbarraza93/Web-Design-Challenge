Summary: Latitude vs. X

The Purpose of this project was to analyze weather changes throughout various regions of the equator. We selected over 500 random cities by utilizing NumPy to find random latitudes and longitudes throughout the world. We then made API calls to the OpenWeatherMap API to retrieve the data needed to analyze data and create a dataframe.

Afterwards, we used matplotlib to plot latitude vs. different weather factors such as temperature, cloudiness, wind speed, and humidity. This website demonstrates the data used for our analysis as well as the scatter plots for each factor, along with my analaysis of the trends and correlations.
